DBpedia Querying App

Run the python file to open a Window that will allow you to do a book search on DBepedia. The search is done using SPARQL queries. 

External modules used:

	-SPARQLWrapper
	-titlecase
	
Known to work:

	- Catch-22
	- Emma
	- Pride and Prejudice
